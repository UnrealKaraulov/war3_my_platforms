Подготовить xenforo 2, установить аддон, установить чистуд бд сервера. Запустить сервер с включенной регистрацией, создать аккаунт для бота, отключить регистрацию, установить боту все права, и запустить сервер.

Далее регистрация будет идти через Xenforo аддон, при создании аккаунта нужно будет указать никнейм на сервере и после подтверждения почты будет создан аккаунт.
