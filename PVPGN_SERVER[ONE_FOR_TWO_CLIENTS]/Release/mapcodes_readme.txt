Формат данного файла:

Пример:
CODE:{stats,dota}NAME:{DotA v6.88i Allstars}PATH:{DotA Allstars 6.88i.w3x}MODES:{ar|sd|rd|cm|ap|}PLAYERS:{5x5|3x3|1x1|}


CODE - код создания карты, если есть "stats," то карта для  статистики, если нет "stats," или вместо "stats," написано "nostats," - карта будет доступна только для /chost /phost (создание карт без статистики / создание карт без статистики - приватная игра)

NAME: Имя карты, будет видно игроку при написании /maplist или /#host команд

PATH: Путь до карты, для ботов, если не найдено !load CODE то будет !map PATH 

MODES: доступные моды для hlc бота, будет дорабатываться

PLAYERS: доступное количество игроков для карты, будет загружаться !Load code_playercount.cfg
